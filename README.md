# CarCost

## Start up Frontend

- `npm install`
- `npm run dev`

## Start up Backend

- `python3 -m venv venv`
- `source venv/bin/activate`
- (Windows Powershell) `venv\Scripts\Activate.ps1`
- `pip install -r requirements.txt`

- `python manage.py runserver`

- `python manage.py migrate`
- `pip freeze > requirements.txt`
